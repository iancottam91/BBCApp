# BBC Test

## Overview:

* A solution to the BBC iPlayer test
* Technologies used:

DEV:
- AngularJS
- PureCSS (cssframework)
- Sass
- Compass

TEST:
- Karma
- Jasime 

## Running the tests:

* open a terminal at */tests
* run npm install to get all the node modules
* run npm test to run the tests using karma
